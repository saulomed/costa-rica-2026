# 🇨🇷 Costa Rica 2026 — Saulo & Lorena

Roteiro interativo da viagem à Costa Rica (09–17 de Abril de 2026).

🌐 **[Ver o site ao vivo →](https://<SEU_USUARIO>.github.io/costa-rica-2026/)**

## Sobre

Página web interativa com:
- 🗓️ Calendário clicável com detalhes dia a dia
- ⚠️ Seção de alertas (ingressos, reservas, dinheiro)
- 📍 Timeline com horários e atividades
- 💰 Estimativa de custos
- 🏨 Opções de hospedagem por dia
- 📱 Design responsivo (funciona no celular!)

## Como atualizar

1. Edite o arquivo `index.html`
2. Faça commit e push para a branch `main`
3. O GitHub Actions faz o deploy automaticamente em ~1 minuto

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Google Fonts (Playfair Display, DM Sans, JetBrains Mono)
- Unsplash (imagem do hero)
- GitHub Pages (hospedagem gratuita)
- GitHub Actions (CI/CD automático)
